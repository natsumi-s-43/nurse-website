# Home Nursing Care Website

This is a website created for a home nursing care service that provides compassionate and professional care to patients in their own homes.  
The site is designed to be clear, accessible, and trustworthy for patients, families, and caregivers.

## Demo
https://natsumi-s-43.github.io/nurse-website/

## Overview

The website introduces home nursing services, explains the type of care provided, and helps users easily find important information.  
Special attention was given to usability and readability for a wide range of users.

## Features

- Clear presentation of home nursing services
- Easy-to-understand layout and navigation
- Responsive design for desktop and mobile devices
- Calm and friendly UI to build trust

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This project was created as a portfolio project to demonstrate:
- Website design focused on accessibility and clarity
- Creating user-friendly layouts for healthcare-related services
- Building responsive websites using fundamental web technologies

## Target Users

- Patients receiving home nursing care
- Family members and caregivers
- Healthcare service providers
